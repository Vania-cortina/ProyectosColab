# Proyectos en Google Colab con Python

El propósito principal de este repositorio es mantener un registro organizado de los proyectos realizados, facilitando su consulta, reutilización y análisis posterior. Cada notebook contiene código, explicaciones y resultados obtenidos durante el desarrollo de las actividades.

---

## Proyectos incluidos

### 1. Programa de reconocimiento de emociones
Este proyecto consiste en el desarrollo de un programa capaz de **identificar emociones a partir de expresiones faciales** mediante el uso de técnicas de visión por computadora y aprendizaje automático.

El sistema analiza imágenes y clasifica las emociones presentes. Entre las emociones que pueden identificarse se encuentran:

- Feliz
- Triste
- Enojado
- Sorprendido
- Neutral

Para su funcionamiento se emplean librerías de Python relacionadas con **procesamiento de imágenes, reconocimiento facial y modelos de clasificación**. El objetivo de este proyecto es explorar el uso de la inteligencia artificial para interpretar emociones humanas a partir de datos visuales.

---

### 2. Análisis de base de datos de suicidios
En este notebook se realiza un **análisis exploratorio de una base de datos relacionada con suicidios**. A través de diferentes consultas y procesamiento de datos se busca identificar patrones y obtener información relevante.

Este análisis tiene un enfoque **académico y estadístico**, permitiendo practicar el manejo de datos y el uso de herramientas de Python para el análisis de información.

## Tecnologías utilizadas

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib / Seaborn
- Librerías de visión por computadora
## Objetivo del repositorio
El repositorio tiene como finalidad servir como **evidencia de aprendizaje y práctica en el uso de Python para análisis de datos e inteligencia artificial**, así como fomentar buenas prácticas en la organización y documentación de proyectos desarrollados en entornos colaborativos como Google Colab.
